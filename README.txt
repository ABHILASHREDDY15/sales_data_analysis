# Sales Data Analysis Project

## Overview
This project analyzes fictional sales data using Excel. The dataset includes order details such as customer name, product, region, quantity, unit price, and total amount.

## Skills Demonstrated
- Data Cleaning
- Pivot Tables
- Sorting & Filtering
- Data Visualization (Charts)

## Insights
- Identified total sales by region.
- Found the top 5 products by sales revenue.

## Tools Used
- Microsoft Excel
